## 🔐 Que Informações Você Precisa?

Antes de começar o deploy, você precisará de:

1. **Token GitHub** (para GitHub API)
   - Criar em: https://github.com/settings/tokens
   - Escopo: `repo`
   - Guardar com segurança

2. **Senha Admin** (para autenticação no app)
   - Escolha uma senha forte e memorável
   - Exemplo: `Senha123!Segura`

3. **Username GitHub**
   - Seu nome de usuário no GitHub
